Polar plot files

CyBot-Plot-Sensor-Scan-Hardcoded-Values.c: Starter script (must rename from .c to .py)

CyBot-Plot-Sensor-Scan-Values-From-File.c: Starter script to read sensor data from a file (must rename from .c to .py)

mock-cybot-sensor-scan.txt: Mock sensor data

mock-cybot-sensor-scan.txt is a text log file that contains mock sensor data. Each row of the log file contains an angle at which the Cybot gets a distance measurement, and the distance it measured. The Cybot collects data scanning from 0 to 180 degrees at 4 degree increments.
